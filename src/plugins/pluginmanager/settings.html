About third-party plugins and recovery

Third-party plugins can stop DashPi from starting if their code or dependencies fail during startup.

Recovery usually means removing the plugin folder over SSH, then restarting the service:

ls ~/DashPi/src/plugins
sudo rm -rf ~/DashPi/src/plugins/PLUGIN_FOLDER
sudo systemctl restart dashpi.service
